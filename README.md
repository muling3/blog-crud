# Golang Blog Application Using microservices

### Services
- `frontend` for rendering the html logic
- `auth` for authenticating users
- `blog` for performing `CRUD` on blogs
- `mailing` for notifying of new blogs `created`, `updated` or `deleted` and also for any errors happening on the system.
  
### Databases
- `postgres`, `mysql` and `mongo db`
  
### Frontend
- `HTML`, `CSS`, `JavaScript` and `tailwindcss`
  
### Other technologies to be used 
- containerization: `docker` and `kubernetes -> minikube`
- Messaging queues: `kafka` and `rabbitmq`
